  • Basics

  • How can I sign up?

    If you are a student or a faculty of IIITA then you will be default signed up, linked to the college Mail ID provided to you. If not, then you cannot sign up.

  • Does it work on the mobile too?

    Not for now, as it is not a published website.

  • Can Quizzes be conducted through this portal?

    This is a demo version and thus this facility has not been provided.

  • Is this software available only on IIITA LAN, just like AVIRAL used to be?

    This cannot be answered as this is a demo website, but if published it could be accessed from anywhere.

  • Account

  • How do I change my password?

    You cannot change this password as your account credentials are your Ldap credentials, that being said the only way to change password would be from the server room.

  • How do I delete my account?

    Only the admin can delete this account, which would be if the person leaves the college.

  • How do I change my account settings?

    For this demo website, the account settings can only be changed by the admin, this is be.

  • Document

  • Can any type of documents be uploaded?

    Most of the documents that are uploaded by faculty of college are images and PDF's or programming files which can be uploaded without any hassle. However, links can also be uploaded.

  • Can I remove a posted document?

    This option is only available if you are a course instructor. If so, then you need to navigate to the subject where the posted document needs to be deleted and you can remove this document by clicking on delete option available on the side of the uploaded material.

  • How do I upload files from my dektop?

    The course instructor just needs to navigate through the website to the course whose materials he needs to upload and click on the upload button, then a dialog box will appear through which he can chose any document in his/her system and upload.

  • How can I add a link?

    For this demo version, you can just add links to a document and upload it under the respective subject. The students can download this pdf and then they would be able to access these links.

  • How can I update attendance of students in real time?

    There is a attendance tab in every user's profile. Any course instructor can navigate to this tab, where a google doc link would be available. Any user can redirect to this link created by admin where the attendance of students can be viewed/updated in real time. However, the option for editing student's attendance will only be available to course instructor.

  • Privacy

  • My files are missing! How do I get them back?

    If the files have been missing from student's account, then he can contact the server room, which will then resolve the issue.

  • How can I access my account data?

    Any user can navigate to the profile tab which is present in every user's dashboard.
